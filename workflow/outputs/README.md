# Workflow Outputs

Folder ini berisi hasil kerja dari agents.

## Structure

| Folder | Agent | Content |
|--------|-------|---------|
| `01-product/` | @workflow/agents/product.md | PRD, user stories, roadmap |
| `02-engineering/` | @workflow/agents/tech-lead.md | Tech specs, architecture, routes |
| `03-tasks/` | @workflow/agents/tech-lead.md | Task breakdowns |
| `04-reports/` | @workflow/agents/qa.md | Test reports |
| `05-deployment/` | @workflow/agents/devops.md | Deployment config |

## File Naming

- Gunakan UPPERCASE untuk file utama: `PRD.md`, `TECH_SPEC.md`
- Gunakan kebab-case untuk tambahan: `database-schema.md`
